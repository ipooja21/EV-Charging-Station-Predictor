## 📌 Overview

**EVI-PlanNet** is an intelligent spatial-predictive system designed to assist urban planners and EV infrastructure developers in identifying **optimal locations** for electric vehicle charging stations across Indian cities.

As EV adoption accelerates, the demand for well-placed charging infrastructure becomes critical. This system uses **Machine Learning** and **geospatial data analysis** to predict the most strategic deployment locations — minimizing range anxiety and maximizing coverage.

> **Goal:** Use data-driven predictions to solve the real-world problem of EV charging infrastructure planning in Indian urban landscapes.

## 🎯 Problem Statement

India's EV adoption is growing rapidly, but charging infrastructure lags behind. Key challenges:

- 🔴 **Uneven distribution** of charging stations in cities
- 🔴 **Range anxiety** among EV users due to poor planning
- 🔴 **No data-driven approach** for station placement decisions
- 🔴 **Wasted investment** on poorly located stations

---

## ✅ Solution

EVI-PlanNet addresses these challenges by:

- 🟢 Using **ML models** trained on urban mobility & demand data
- 🟢 Analyzing **geospatial patterns** to find high-demand zones
- 🟢 Providing **priority scores** for each potential location
- 🟢 Offering a **web interface** for planners to explore predictions

---

## ⚙️ How It Works

```
1. 📥 Input urban data (population, traffic, EV usage)
2. 🔧 Preprocess & engineer geospatial features
3. 🤖 ML model predicts location suitability scores
4. 🌐 Flask API serves predictions on request
5. 📊 Dashboard visualizes recommended locations
6. 📍 Planners use output for infrastructure decisions
```

---

## 🎯 Challenges Solved

- **Sparse Data** → Feature engineering from available urban datasets
- **Geospatial Complexity** → Location encoding for ML compatibility
- **Multi-factor Decision** → Combined scoring from multiple urban variables
- **Scalability** → Flask backend handles multiple city requests
